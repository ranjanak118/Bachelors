# Accomodation-for-Bachelors
Android applications that caters to the rental requirements of bachelors.

SEQUENCE OF PAGES :

Common page -> Sign up -> Register -> Login
            -> Login -> Dashboard&Search -> Navigation bar
            
Navigation bar -> User Profile
               -> Dashboard&Search
               -> Chat
               -> Add Property
               -> Wallet
               -> Notifications
               -> Suggestions
               -> Sign out 
               
Sign out -> Login
